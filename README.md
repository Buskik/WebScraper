<h2> Setup </h2>
 

1. Install node dependencies.
```
npm i
```
2. Run app.
```
node app
```
3. Open localhost in browser
```
http://localhost:3000/ //HTML page
http://localhost:3000/api/scrape //API Endpoint
```

### Dependencies
> Express.js, Axios, JSDOM.
